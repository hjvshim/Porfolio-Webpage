석사 학위 졸업 논문

[ 논문 제목 ]
Obstacle Avoidance using Density and Distance Aware Compressed Grid Map through Deep Q-Network

[ 논문 요약 ]
인공지능 기반의 장애물 회피에 관한 연구에서 주로 사용되는 강화학습 모델인 Deep-Q-Network(DQN)는 실시간으로 에이전트가 보는 Scene을 처리하기 때문에 학습 시간이 매우 길어진다는 문제가 있습니다. 이 점에 착안하여 본 논문에서는 Input으로 이용되는 이미지를 sub grid로 나누어 더 작은 이미지로 압축시켜 학습 시간을 줄이면서 정확도를 유지하는 Obstacle Density-aware Compressed Grid Map을 제안합니다.
이는 처음 DQN에서 받는 에이전트가 바라보는 뷰의 이미지들을 그레이스케일로 변환하여 일정 크기로 그리드화하여 장애물을 식별해 맵으로 변환시킵니다. 이 때, 에이전트의 위치에서부터 장애물까지의 거리차를 이용하여 색상을 가까울수록 흰색에 가깝도록 레벨링하는 Distance-based Gray Scale Leveling 또한 적용했습니다.
맵의 사이즈를 10x10 이하로 줄이기 위해 sub grid로 나누어 각 sub grid 값들의 합을 정규화하여 10x10 맵에 맵핑시켜 압축시킵니다. 에이전트는 전방의 가장 가까운 곳을 위주로 움직이다가 근방이 모두 막혔을 경우 바깥쪽으로 나아가게 되며, 압축된 맵을 통해 장애물이 적은 방향을 향해 나아가게 됩니다. 다양한 Grid map size 및 Sub-grid size로 실험을 한 결과, Training time은 정확도의 감소 없이 기존 DQN보다 약 74% 빨라졌습니다.

스마트 센싱 세상 만들기

HCI 연구실에서 공부할 때 진행했던 스마트폰 USB 센서 Plug&Play를 위한 통합 App 구현 프로젝트입니다. 본 프로젝트에서는 하나의 App으로 다종의 센서를 처리하는 Open Unified App Architecture를 제시합니다. 다양한 기기에 적용이 가능하도록 WebApp으로 구현되었으며, 스마트폰에 USB 센서를 부착하면 관련 센서의 Device Driver를 다운로드하여 웹 서버에서 센서에 알맞는 웹 페이지를 다운 받아 교체합니다.

이 프로젝트로 연구실 선배님들과 함께 2012년 12월에 임베디드 소프트웨어 경진대회에 참가하여 우수상(NIPA 원장상)을 수상하였습니다.
